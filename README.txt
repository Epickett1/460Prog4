i. Compilation and execution instructions

Compilation:
Compile menu.java (javac menu.java)
Add jdbc8 classpath
	export CLASSPATH=/usr/lib/oracle/19.8/client64/lib/ojdbc8.jar:${CLASSPATH}

Execution
java menu <YOUR USERNAME> (excluding @oracle.aloe) <YOUR PASSWORD>
